字符串的匹配（BF、BM）虽然在笔试或者机试上可能不会考察
但是在面试的时候，比如三面或者四面的时候，面试官完全会问到这种题，回答相应的思路或者原理


字符串的匹配BF算法，Brute Force算法（Force有点暴力破解的意思）
    最坏的时间复杂度为O(m*n)，但是在工程实践中，一般都接近于O(m+n)
    并且容易理解，代码实现相对比较简单，所以很多工程实践也是用的这种算法
    空间复杂度：没有引入额外的空间，所以为O(1)
    String.indexOf()方法就是基于该思想完成的
